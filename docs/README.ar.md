## ⚡ ما هو Qwen Orchestrator؟

**Qwen Orchestrator** هو امتداد قوي مبني حصريًا لـ **[Qwen Code](https://github.com/QwenLM/qwen-code)** — أداة CLI للبرمجة AI مفتوحة المصدر من Alibaba. يحوّل مساعد برمجة AI واحدًا إلى **فريق منسق من 24 وكيل AI متخصص** يعملون معًا كقسم تطوير برمجيات محترف.

### لماذا Qwen Orchestrator؟

تخيل وجود فريق هندسي كامل في متناول يدك: **Commander** ينسق، و**Planner** يصمم البنية، و**مطورو الواجهة الأمامية والخلفية** يبرمجون بالتوازي، و**Reviewer** يحمي الجودة، و**QA Engineer** يختبر، و**Cybersecurity Engineer** يدقق الثغرات، و**DevOps Engineer** ينشر — بالإضافة إلى 16 متخصصًا إضافيًا، جميعهم منسقون تلقائيًا.

### مبني حصريًا لـ

**[Qwen Code](https://github.com/QwenLM/qwen-code)** من [QwenLM / Alibaba](https://github.com/QwenLM) — أداة CLI لمساعد البرمجة AI مفتوحة المصدر تدعم مزودي LLM متعددين. Qwen Orchestrator هو **امتداد مبني من المجتمع** وليس تابعًا أو معتمدًا من Alibaba.

> ⚠️ **هذا الامتداد يعمل فقط مع [Qwen Code CLI](https://github.com/QwenLM/qwen-code)**. ليس أداة مستقلة، وليس امتداد VS Code، وليس متوافقًا مع مساعدي البرمجة AI الآخرين.

## 🎯 بداية سريعة

```bash
/orchestrator إنشاء REST API لإدارة المستخدمين
/orchestrator بناء موقع تجارة إلكترونية متجاوب
/plan تصميم مخطط قاعدة بيانات لمدونة
/review فحص جميع متحكمات المصادقة
/test تشغيل جميع اختبارات المشروع الحالي
/deploy نشر التطبيق إلى الإنتاج
```

أمر `/orchestrator` هو نقطة الدخول الرئيسية. ينشط الفريق الكامل المكون من 24 وكيلاً:

1. **توضيح** → يطرح أسئلة محددة إذا كان هناك غموض
2. **اكتشاف** → يفحص المشروع ويكتشف مجموعة التقنيات
3. **تخطيط** → يحلل المهمة إلى مراحل مع مجموعات تنفيذ متوازية
4. **تنفيذ** → يشغل الوكلاء المتخصصين في وقت واحد
5. **تحقق** → Reviewer + QA يؤكدان أن كل شيء يعمل
6. **تسليم** → ملخص لما تم بناؤه وتغييره والأدلة

```bash

```

```bash
git clone https://github.com/Omar-Obando/qwen-orchestrator.git
```

أضف إلى الإعدادات (`~/.qwen/settings.json`):

```json
{
  "extensions": ["/المسار/الكامل/qwen-orchestrator"]
}
```

### خادم أدوات MCP (اختياري)

```bash
cd qwen-orchestrator && npm install && npm run build
```
