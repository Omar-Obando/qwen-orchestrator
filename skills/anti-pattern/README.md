# Anti-Pattern

Detection and prevention of common code anti-patterns.

## Overview

This skill helps identify and eliminate common anti-patterns in code including mockups, placeholders, N+1 queries, and dead code.

## Key Topics

- Mockup and placeholder detection
- N+1 query prevention
- Dead code identification
- Code smell patterns
- Refactoring strategies

## When to Use

- Code review processes
- Refactoring existing code
- Preventing technical debt
- Improving code quality
